Elevator Speech

Despite being connected to hundreds of friends on Facebook, many people report feeling isolated. Asking friends to spend time with you one by one (via phone, SMS, e-mail, etc...) is feasible but it is painfully slow and cumbersome. Our system is designed to make it easy for people to send open invitations to their close group of friends: “I will be working out in Dillon Gym from 3-4pm today. Click here to join me”. Our system is for people who don't want to crowd their friends' inboxes with requests to hang out and don't want to organize large events, but who would like to have some company while they're working, going to the gym, or making dinner. We want to solve the problem of spontaneous gatherings using new mobile technologies.

How to Access a Running Version of the System

We will give you an iPhone with the application installed so that you can test it out. The source code is also available on github. Note that if we actually deploy the application, we will change all secret keys and functions from those in the public versions to prevent hackability.

Please contact prea@princeton.edu and azheng@princeton.edu if something goes terribly wrong.
